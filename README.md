# CelebDatingSampling

Data Science Project on web scraping and sampling.

# Project
To obtain a sampleof 130 celebrities using two methods or comparison.
- Method 1: Snowball sampling: starting with one celebrity, we sample the next celebrity from a list of who've the first celebrity has dated. We recursively do this until our sample size is 130. I our project, we tried this with Matthew McConaughey, Cillian Murphy, and Emma Waston.
- Method 2: Alphabet sampling: For each letter of the alphabet we take the 5 most popular celebrities on the website.

# Data
The data for this project was sourced/scraped from the website https://www.whosdatedwho.com
